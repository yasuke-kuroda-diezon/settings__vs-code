# 修正の概要を箇条書きで生成する。
1. `git add .`を実行する。
2. `git diff --cached`を実行する。
3. 取得した変更内容から主要な修正点を1ファイル1行で箇条書き形式でまとめる。
```
- ファイル名1
  - 修正内容の概要1
- ファイル名2
  - 修正内容の概要2
- ファイル名3
  - 修正内容の概要3
```
